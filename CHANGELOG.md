# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1

- Initial release.
- Added reflog webview with compact left/right selection controls.
- Added compare actions for all changed files and active file only.
- Added row click behavior to compare an entry with its previous reflog entry.
- Added timed status-bar notifications for extension feedback.
- Added configurable auto-refresh interval (`reflogDiff.refreshIntervalMs`).
